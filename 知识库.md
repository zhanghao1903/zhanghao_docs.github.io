    我自己有几台设备，一台是Mac mini m2 8g+256g,一台是win mini主机 AMD 7940HS 32g+1T，还有一台笔记本m2 pro 16g + 512g.
    在github看了两周的文档，我清楚知道，现在很多AI项目对win平台的支持并不好，也不及时，所以就淘汰了。
    我主要还是想本地部署GUI，所以最终还是选择了MacBook pro, 虽然内存也不大，但总归是比mini 大一点。
    本地环境搭建：
    openhand对macos的支持还是挺好的，可以一键部署。不过在此之前，需要先安装uv。
    1. 安装 uv
    curl -LsSf https://astral.sh/uv/install.sh | sh
    2. 安装openhand
    uv tool install openhands --python 3.12
    总体上比较顺利，就是安装uv的时候遇到一些权限的问题，也很快解决了。openhand安装也很快也方便。第一次启动时遇到了点问题，重新启动就好了。
    openhand cloud 也集成了多家LLM provider的能力。最关键的是：
    Prices listed reflect provider rates with no markup, sourced via LiteLLM’s model price database and provider pricing pages.
    openhand 不收任何手续费，起初我觉得没啥，后面知道openrouter 要收5% -8%的手续费的时候，我直呼openhand 良心。
    我使用的是openhand cloud作为LLM provider。基本上openAI、Claude、Gemini的主流模型都能使用，其默认推荐就是Claude code，我一开始对这些模型
并不是很了解，所以使用的默认选项。
